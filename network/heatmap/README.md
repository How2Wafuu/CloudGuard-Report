# Heatmap

To ensure seamless connectivity and high-speed data access across the organization, we conducted a wireless site survey simulation using the HPE Aruba AP-635. This section visualizes the signal propagation across the 2.4 GHz, 5 GHz, and 6 GHz (Wi-Fi 6E) frequency bands.

### Frequency Band Comparison

* **2.4 GHz:** Optimized for maximum coverage and wall penetration, ideal for IoT devices and basic mobile connectivity.
* **5 GHz:** Offers high-speed throughput with moderate range, suitable for high-bandwidth applications.
* **6 GHz (Wi-Fi 6E):** Provides ultra-low latency and multi-gigabit speeds with zero interference from legacy devices, leveraging the newly opened 6 GHz spectrum.
