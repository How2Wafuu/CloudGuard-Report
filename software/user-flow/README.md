# User Flow

CloudGuard AI manages two distinct but complementary user experiences tailored to different roles within an organization.&#x20;

[client-onboarding-and-response-flow.md](client-onboarding-and-response-flow.md "mention")

The first is the SME Client experience, which is designed to provide clear, high-level asset visibility, helping small and medium-sized enterprises understand what they have in their environment, where it is located, and how it is performing from a security standpoint.&#x20;

[soc-analyst-flow.md](soc-analyst-flow.md "mention")

The second is the SOC Analyst experience, which focuses on in-depth investigation, threat detection, and rapid containment, providing security professionals with the tools and context they need to analyze suspicious activity, respond to incidents, and continuously improve the organization’s security posture.
